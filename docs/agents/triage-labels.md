# Triage labels

| Role                          | GitHub label      |
| ----------------------------- | ----------------- |
| Needs evaluation              | `needs-triage`    |
| Waiting for information       | `needs-info`      |
| Ready for an agent            | `ready-for-agent` |
| Requires human implementation | `ready-for-human` |
| Will not be actioned          | `wontfix`         |
