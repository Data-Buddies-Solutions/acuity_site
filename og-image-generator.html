📸 How to Generate OG Image

  1. Open this file in Chrome or Firefox
  2. Press F11 for fullscreen
  3. Zoom to 100% (Ctrl/Cmd + 0)
  4. Right-click the dark image and select "Inspect"
  5. In DevTools, right-click the .og-image element
  6. Select "Capture node screenshot"
  7. Save as og-image.png
  8. Move to /public/og-image.png

Or use a screenshot tool to capture the 1200×630 dark image.
